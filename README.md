# readie. | build your knowledge web

the official landing page and privacy policy for the **readie.** app. a minimalist, ai-powered reading hub designed for focused learning and knowledge building.

## overview

this repository contains the public-facing site for **readie.**, including the main landing page showcasing the app's unique features and the mandatory privacy policy for google play compliance.

### core features of readie.
- **ai pre-reading**: let ai skim articles before you dive deep.
- **knowledge network**: visualize how your atomic insights connect across sources.
- **rich content support**: native rendering for code and complex math formulas.
- **smart notes**: take entity-based notes that automatically grow your web.
- **distraction-free focus**: a pure, clean interface for deep reading.

## site structure

- `index.html`: the main entry point showcasing the app's vision and features.
- `privacy.html`: legal privacy policy for transparency and store compliance.
- `styles.css`: shared minimalist design tokens and responsive layout.
- `assets/images/`: high-resolution screenshots highlighting the app's core functionality.

## github pages deployment

to host this site using github pages:

1. navigate to **settings → pages** in this repository.
2. set **source** to `deploy from a branch`.
3. select the `main` branch and the `/ (root)` folder.
4. save the configuration.

your landing page will be available at:
`https://<your-username>.github.io/readie-page/`

your privacy policy will be available at:
`https://<your-username>.github.io/readie-page/privacy.html`
