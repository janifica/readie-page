# readie. landing page

this is the public landing site and privacy policy for readie.

## about

readie. is a minimal reading hub designed to help you capture, organize, and revisit important ideas from web articles, videos, and links. it focuses on deep retention and distraction-free reading, powered by intuitive ai assistance.

## pages

- **landing page**: a showcase of readie's core features—ai pre-reading, knowledge web visualization, and rich content support.
- **privacy policy**: our commitment to user privacy and data security.

built by independent developers with a focus on simplicity and privacy.
